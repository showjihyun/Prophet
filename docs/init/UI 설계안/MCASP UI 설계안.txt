 **Miro Fish** UI를 보면 일반적인 **그래프 분석 툴(예: Gephi)**과 완전히 다른 철학을 가지고 있습니다.

핵심 특징은 다음입니다.

```
Graph 분석 UI ❌
"살아있는 사회 시뮬레이션" UI ✔
```

즉 **네트워크 분석 툴 느낌이 아니라 “AI 사회를 관찰하는 공간”**입니다.

그래서 MCASP UI도 다음처럼 바꾸는 것이 좋습니다.

---

# 1️⃣ Miro Fish 스타일 UI 철학

기존 설계

```
Graph Tool
```

Miro Fish 스타일

```
AI Social World
```

핵심 UX

1️⃣ 살아있는 Agent
2️⃣ 자연스러운 움직임
3️⃣ 대화 흐름
4️⃣ 확산 애니메이션

즉

```
Graph → Social Ecosystem
```

---

# 2️⃣ Miro Fish 스타일 메인 화면 구조

그래프를 크게 두되 **“생태계 느낌”**으로 표현합니다.

```
+------------------------------------------------------------------+
| MCASP Simulation Studio                                          |
| Project | Scenario | Run | Pause | Speed                         |
+------------------------------------------------------------------+

+------------------+--------------------------------------+--------+
|                  |                                      |        |
|                  |                                      |        |
| Communities      |        AI SOCIAL WORLD               | Metrics|
|                  |                                      |        |
|                  |                                      |        |
+------------------+--------------------------------------+--------+

+------------------------------------------------------------------+
| Timeline + Viral Diffusion                                       |
+------------------------------------------------------------------+

+------------------------------------------------------------------+
| Agent Conversations / Expert Analysis                            |
+------------------------------------------------------------------+
```

그래프 이름도

```
Agent Network
```

이 아니라

```
AI Social World
```

---

# 3️⃣ Agent Visualization (Miro Fish 스타일)

Agent를 **노드가 아니라 캐릭터처럼 표현**합니다.

```
    ● Agent12

 ● Agent45      ● Agent22

       ● Agent90
```

Agent 상태

| 상태     | 표시    |
| ------ | ----- |
| 메시지 작성 | 말풍선   |
| 영향력 증가 | Glow  |
| 확산 시작  | Pulse |
| 논쟁 발생  | 빨간 파동 |

---

# 4️⃣ Community Visualization

커뮤니티를 **클러스터가 아니라 “섬”처럼 표현**

```
   Community A

    ● ● ●
   ● ● ●

--------------------

   Community B

   ●●●●●●
  ●●●●●●
```

커뮤니티 간 연결

```
A ---- B ---- C
```

# 5️⃣ Viral Diffusion Animation

Miro Fish 핵심

```
메시지 확산 애니메이션
```

예

```
Agent34

   ↓

Agent56

   ↓

Agent77
```

그래프에서는

```
Agent34 ))) Agent56 ))) Agent77
```

처럼 파동으로 표시
