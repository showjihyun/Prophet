좋습니다. 여기서는 **Stanford Generative Agent 구조를 확장한 “차세대 Agent Memory + Behavior Architecture”**를 설계합니다.
핵심 목표는 다음입니다.

* 1000+ Agent 스케일
* Multi-community 사회 모델
* LLM 비용 최소화
* Emergent behavior 생성
* GraphRAG 기반 기억

기본 참고 연구는
Generative Agents: Interactive Simulacra of Human Behavior 입니다.
하지만 이 구조는 **Stanford 모델보다 더 발전된 5-Layer 구조**입니다.

---

# 1️⃣ 기존 Stanford Generative Agent 구조

Stanford 모델

```text
Observation
   ↓
Memory Stream
   ↓
Reflection
   ↓
Planning
   ↓
Action
```

문제

1️⃣ LLM 호출이 너무 많음
2️⃣ 사회 네트워크 모델 없음
3️⃣ 영향력 모델 없음
4️⃣ 확산 모델 없음

따라서 MCASP에서는 **완전히 다른 구조**가 필요합니다.

---

# 2️⃣ MCASP Agent Architecture

새로운 구조

```text
Perception Layer
        ↓
Memory Layer (GraphRAG)
        ↓
Emotion & Cognition Layer
        ↓
Decision Layer
        ↓
Action Layer
        ↓
Social Influence Layer
```

총 **6 Layer 구조**

---

# 3️⃣ Agent Memory Architecture

Agent 기억은 **3가지 타입**

```text
Episodic Memory
Semantic Memory
Social Memory
```

---

## Episodic Memory

Agent 경험

예

```text
"t=12 influencer mentioned product"
"t=15 friend recommended phone"
```

구조

```text
M_episode = {
 event,
 timestamp,
 emotional_weight
}
```

---

## Semantic Memory

Agent 지식

예

```text
"brand reputation"
"product features"
```

구조

```text
M_semantic = {
 concept,
 belief_strength
}
```

---

## Social Memory

사람 관계 기억

```text
M_social = {
 agent_id,
 trust_score,
 influence_weight
}
```

---

# 4️⃣ GraphRAG Memory

Memory는 그래프로 저장

Node

```text
agent
event
belief
community
```

Edge

```text
experienced
trusts
discussed
influenced
```

예

```text
Agent_24 → experienced → campaign_ad
Agent_24 → trusts → Agent_8
Agent_8 → influenced → Agent_24
```

이 구조는 사실상

**Social Knowledge Graph**

# 5️⃣ Memory Retrieval

Agent 행동 전에 Memory 검색

score

```text
Score =
recency +
relevance +
emotion_weight +
social_importance
```

수식

```text
Score =
α*recency
+ β*relevance
+ γ*emotion
+ δ*social_weight
```

상위 K memory

```text
TopK = 10
```

# 7️⃣ Cognition Layer

Agent 판단

Stanford 모델은 전부 LLM 사용

MCASP

**Hybrid cognition**

```text
Rule Engine
+ Influence Model
+ LLM reasoning
```

---

### Rule Engine

빠른 판단

예

```text
if trust(friend) > 0.7
→ share_probability += 0.2
```

---

### Influence Model

```text
social_pressure =
Σ influence_j * trust_ij
```

---

### LLM Reasoning

복잡한 판단

예

```text
Should I recommend this product?
```

LLM 호출은

```text
10~20% agent only
```

---

# 8️⃣ Decision Model

Agent 행동 확률

행동

```text
ignore
like
share
comment
purchase
```

확률

```text
P(action) =
softmax(
interest +
social_pressure +
emotion +
memory_weight
)
```

---

# 9️⃣ Social Influence Model

Agent 간 영향

그래프

```text
G(V,E)
```

Edge weight

```text
W_ij =
trust_ij * interaction_freq
```

확산

```text
P(i → j) =
influence_i
* trust_ij
* emotion_factor
```

---

# 🔟 Behavior Loop

Agent 행동 loop

```text
1 perceive environment
2 retrieve memory
3 update emotion
4 compute social pressure
5 run cognition
6 choose action
7 update memory
```
