좋습니다. 여기서는 **MCASP의 핵심 엔진인 Social Diffusion Engine (Viral Cascade Algorithm)**을 **연구/논문 수준으로 설계**합니다.
이 엔진의 목표는 **현실 SNS에서 나타나는 바이럴, 확산, 논쟁, 붕괴 현상**을 재현하는 것입니다.

기본적으로 다음 연구들을 기반으로 합니다.

* Viral Marketing for the Real World
* Independent Cascade Model
* Linear Threshold Model

하지만 MCASP에서는 이들을 **LLM 기반 Agent와 결합한 Hybrid Diffusion Engine**을 설계합니다.

Propagation 예

```python
def propagate(agent_i, agent_j):

    influence = agent_i.influence
    trust = trust_matrix[i][j]

    emotion = agent_i.emotion["excitement"] - agent_j.emotion["skepticism"]

    p = influence * trust * emotion

    return random.random() < p
```
