MCASP 플랫폼에서 **“마케팅 캠페인 확산 시뮬레이션”**은 가장 직접적으로 돈이 될 수 있는 핵심 유즈케이스입니다.
단순 SNS 확산 모델이 아니라 **Agent 기반 사회 시뮬레이션 + 전문가 Agent 분석 + Multi-community 영향 분석**까지 포함하면 기업들이 실제로 사용할 수 있는 수준이 됩니다.

아래는 **PRD 수준의 상세 시나리오 설계**입니다.

---

# 📊 Marketing Campaign Diffusion Simulation

## 1️⃣ 목적

기업 또는 기관이 **마케팅 캠페인, 제품 출시, 메시지 전략**이
**사회 집단에서 어떻게 확산되는지 사전에 시뮬레이션**하는 플랫폼

기존 마케팅 툴
→ 단순 SNS analytics

MCASP
→ **가상 사회에서 캠페인을 먼저 실행**

---

# 2️⃣ 시뮬레이션 구조

## Multi-Community 구성

| Community | Agent 수 | 특징         |
| --------- | ------- | ---------- |
| A         | 100     | 얼리어답터      |

| C         | 200     | 회의적인 소비자   |

총 Agent

# 3️⃣ Agent 행동 모델

Agent는 다음 특성을 가짐

### Personality

```
openness
skepticism
trend_following
brand_loyalty
social_influence
```

### Emotional State

```
interest
trust
skepticism
excitement
```

### Memory

Agent는 다음을 기억

```
- 광고 exposure
- 친구 추천
- 전문가 리뷰
- 과거 브랜드 경험
```

# 6️⃣ 확산 모델

확산은 다음 요소로 결정

```
influence_score
trust_network
community_structure
emotion_state
```

간단한 확산 함수 예

```
share_probability =
influence * trust * excitement
```
